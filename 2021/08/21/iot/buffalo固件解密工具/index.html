buffalo固件解密工具

  • 源码：https://gitlab.nic.cz/turris/openwrt/-/tree/test/tools/firmware-utils/src
  • 步骤：https://forum.archive.openwrt.org/viewtopic.php?id=48520
    • Get a copy of the Buffalo firmware. I got mine from here.
    • Unpack the .zip file. There will be a big binary .enc file in it. This is the encrypted firmware.
    • Strip off the first section of the encrypted firmware – up to the second “start” string. In my firmware, the first section was 208 bytes, so this did it: dd if=encrypted-fw.enc of=crypted-fw.enc bs=208 skip=1
    • Find the Buffalo encrypted-firmware decrypting code in the OpenWRT source tree. It’s somewhere under …/tools/firmware-utils/src/…
    • Copy buffalo-enc.c, buffalo-lib.c and buffalo-lib.h somewhere safe, like /tmp
    • Compile the decryptor: gcc -o bufdec buffalo-enc.c buffalo-lib.c
    • Use the decryptor to decrypt the beheaded, encrypted firmware: ./bufdec -d -i crypted.enc -o buffalo-fw.img
  • 实例
    • ：./buffalo-enc -d -i wsr2533dhp_jp_106 -o wsr2533dhp_jp_106_decrypted.bin -O 200 -l
    • 注意：要手动去掉第一个头（第二个start前）或者-O指定偏移（二者是相同效果）、加-l选项

参考：https://modemizer.wordpress.com/2015/08/05/restoring-the-original-buffalo-firmware-on-the-wbmr-hp-g300h/
