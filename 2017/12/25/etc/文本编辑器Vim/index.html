Linux-文本编辑器Vim

发表于 2017-12-25 更新于 2022-02-16 分类于 其它

文本编辑器Vim

简介

  • 没有菜单，只有命令
  • 三种工作模式
    • 命令模式
      • 打开文件后的默认模式
      • 输入会被当作命令而非字符
      • wq退出
    • 插入模式
      • 命令模式下a、i、o进入
      • ESC退出至命令模式
    • 编辑模式
      • 命令模式下冒号进入
      • 输入命令，如设置行号set nu
      • 命令以回车结束运行

常用操作

  • 插入命令（命令模式进入插入模式
    • a：在光标所在字符后插入
    • A：光标所在行尾
    • i：光标所在字符前
    • I：光标所在行首
    • o：光标下插入新行
    • O：光标上插入新行
  • 定位命令（命令和编辑模式下
    • ：set nu—设置行号
    • ：set nonu—取消行号
    • gg—到第一行
    • G—到最后一行
    • nG—到第n行
    • ：n—到第n行
    • $—移至行尾
    • 0—移至行首
  • 删除命令
    • x：删除光标所在处字符
    • nx：删除光标所在处后n个字符
    • dd：删除光标所在行，ndd删除n行
    • dG：删除光标所在行到文件末尾内容
    • D：删除光标所在处到行尾内容
    • ：m，nd—删除m行到n行
  • 复制、剪切、粘贴
    • yy：复制当前行
    • nyy：复制当前行以下n行
    • dd：剪切当前行
    • ndd：剪切当前行以下n行
    • p、P：粘贴在所在行的行下或行上
  • 替换或取消命令
    • r：取代光标所在处字符
    • R：从光标所在处开始替换字符，ESC结束
    • u：取消上一步操作，即撤销
  • 搜索或搜索替换
    • /+string：搜索指定string
      • ：set ic—搜索命令前执行，忽略大小写（noic取消
      • n：下一个
    • ：%s/old/new/g——全文替换指定字符串（%s表示全文替换
    • ：m,ns/old/new/g——在一定范围内替换指定字符串
      • 起始行，终止行 s/要替换的字符串/新的字符串/g或者c
      • g不询问，c询问以确认是否替换
  • 保存和退出命令
    • ：w——保存修改
    • ：w newfilename——另存为新文件（可使用文件路径，如/etc/backup.txt
    • ：wq——保存修改并退出
    • ZZ：快捷键，保存修改并退出
    • ：q！——不保存修改退出
    • ：wq！——保存修改并退出
      • 文件所有者及root可使用
      • 强行保存，如root打开一个只读文件，使修改生效时

使用技巧

  • 文件中导入其它
    • r +文件名：将指定文件的内容导入到此文件中
    • ！+ 命令：不退出vim，执行系统命令
    • r + ！+ 命令：不退出vim，执行命令并将其结果导入此文件中
  • 定义快捷键
    • ：map+快捷键+触发的命令
    • 举例
      • 添加行注释
        • ：map ^P I#
        • Ctrl+v+p会生成ctrl+p的命令，即^P
        • I：跳到行首并进入插入模式
        • #：添加注释符
        • ：退出插入模式至命令模式
        • ctrl+p即可注释当前行
      • 取消当前行注释
        • ：map ^B 0x
        • Ctrl+v+p会生成ctrl+p的命令，即^P
        • 0: 跳到行首（不进入插入模式，同I不同
        • x：删除光标处第一个字符（#一般是每行的第一个字符
        • ctrl+b会取消行注释
      • 插入邮箱等常用信息
        • ：map ^H i+邮箱等+
        • I：当前光标处进入插入模式
        • 添加自定义常用信息
        • esc：退出至命令模式
  • 连续行注释
    • 应用到vim替换命令的格式（：m,ns/old/new/g
    • 添加多行注释
      • ：m,ns/^/#/g
      • m-n行
      • ^：表示行首这个特殊字符
      • #：替换为注释符#（也可以为其它，如//，但需要转义\/\/
    • 取消多行注释
      • ：m,ns/^#//g
      • ^#：将行首的#替换为空，即去掉
      • 一定加^，否则不是行首的#也会被删
  • 替换
    • ：ab mymail 123123123@163.com
    • ab：替换命令，本意是输入a时，回车或空格后变为b
    • 如输入mymail，空格或回车后变为我的自定义邮箱
    • 编辑模式下定义替换的信息
    • 插入模式下使用
  • 配置文件问题
    • 上述操作仅本次vim使用时有效
    • 若要永久生效，需保存到vim的配置文件
      • root用户：/root/.vimrc
      • 其它：/home/用户名/.vimrc
    • 针对编辑模式下的命令，即冒号开头的
    • 保存到配置文件时，不需要带冒号
    • 一条命令占一行
    • 如永久显示行号，文件内容为“set nu”（不带冒号，且占一行
