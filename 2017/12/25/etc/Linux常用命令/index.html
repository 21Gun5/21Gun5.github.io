Linux-Linux常用命令

发表于 2017-12-25 更新于 2022-02-16 分类于 其它

Linux常用命令

网络命令

  • write命令
    • write+用户名
    • 给在线用户发信息，ctrl+d保存结束
    • 前提是通信双方登录到同一主机
    • w查看当前系统在线用户
  • wall命令
    • wall+待发送消息
    • 发广播信息
    • 目标是当前在线所有用户（包括发信者本身也会收到
  • ping命令
    • 格式：ping +选项+IP地址
      • 不加：无限次发送ping包
      • c参数：指定发送次数
    • 功能：测试网络连通性
    • 输出结果关注丢包率是否高
  • ifconfig命令
    • ifconfig+网卡名称+IP地址
      • 直接ifconfig显示所有网卡信息（最常用
    • 查看和设置网卡信息（常用来看本机IP
  • mail命令
    • mail：查看邮件
      • 进入交互界面
      • 直接id号，查看相应邮件
    • mail+用户名：给xxx发邮件
      • ctrl+d结束输入并发送
    • 针对于当前系统的用户，非163等邮件发送
    • 比write高级，目标用户不在线时也可发送，待其上线后提示
  • last命令
    • 直接last
    • 列出目前与过去登录系统的用户信息
    • 也会显示系统何时重启，是重要的日志记录
  • lastlog命令
    • last -u + UID
      • 不加选项，显示所有用户
    • 检查某特定用户上次登录的时间
  • traceroute命令
    • traceroute + 主机域名
    • 显示数据包到主机间的路径，即追踪路由
  • netstat命令
    • netstat+选项
      • 常用选项
        • t：tcp协议
        • u：udp协议
        • l：监听
        • r：路由
        • n：显示IP地址和端口号
      • 例如
        • netstat -tuln：查看本机监听端口
        • netstat -an：查看本机所有网络连接（与tuln不同，可查看正在连接的网络程序
        • netstat -pantu（kali教程中
        • netstat -rn：查看本机路由表（可查看网关
    • 显示网络相关信息
  • setup命令
    • 直接setup即可
    • 配置网络
    • redhat系列专有

关机重启命令

  • shutdown命令
    • shutdown+选项+时间
      • c：取消前一个关机命令
      • h：关机
      • r：重启
    • 举例
      • shutdown -h now：立即关机
      • shutdown -h 20:30 指定具体时间关机
      • shutdown -吃：取消上一关机指令
      • 同样r重启也可以立刻或指定具体时间
    • 关机或重启前，尽量关闭其它服务（尤其是服务器，硬盘等处于高速运转过程，贸然操作可能出错
    • 其它
      • 其它关机命令：halt、poweroff、init 0
      • 其它重启命令：reboot、init 6
      • 关机或重启推荐shutdown
  • 系统运行级别
    • 0:关机
    • 1:单用户（启动最小核心程序做系统修复，类似win中安全模式；仅root登录，故单用户
    • 2:不完全多用户，不含NFS服务（不含网络服务
      • NFS为网络文件服务，用户Linux间文件传输
      • 比较方便，但存在安全隐患，不建议使用
    • 3:完全多用户（即字符界面
    • 4:未分配
    • 5:图形界面
    • 6:重启
    • 其它
      • init命令选择具体运行级别
      • /etc/inittab文件来修改系统默认运行级别
      • runlevel：查询系统运行级别（第一个数字是前一个，第二个是当前的
  • 退出登录命令
    • logout

用户管理命令

  • useradd和passwd
    • useradd+用户名
      • 添加新用户
    • passwd+用户名
      • 为用户设置密码
      • 新用户必须设置密码后才可以登录，才可正常使用
  • who：查看登录用户信息
  • w：查看登录用户详细信息

帮助命令

  • man命令：详细帮助信息
    • 简介
      • manual，格式：man + 命令或配置文件
      • 浏览过程类似more或less，回车换行、空格翻页、q退出、\搜索等
    • 可查看某命令的帮助信息
      • 一看：此命令具体功能
      • 二看：此命令某个选项或参数的作用
    • 可以查看配置文件的帮助信息
      • man + 配置文件名称，不可加绝对路径
      • 一看：此配置文件作用；
      • 二看：此配置文件格式
    • man passwd相关
      • 即存在命令又存在配置文件，优先显示命令的帮助信息
      • 多种帮助类型
        • 1:命令的帮助
        • 5:配置文件的帮助
      • man 5 passwd：查看配置文件的帮助
  • 查看简要帮助信息
    • whatis命令
      • 简单查看命令的作用
    • apropos命令
      • 简单查看配置文件的作用
      • 同样，不加绝对路径，只加配置文件名即可
  • 只查看命令的某参数
    • -h
    • —help
    • 很多命令通用
  • help命令：内置命令的帮助
    • 获取shell内置命令的帮助信息
    • shell内置命令，找不到绝对路径，是内置的
    • 用man无法获取，用help，如help cd
  • info命令：与man大同小异

压缩解压命令

  • gzip命令
    • 压缩：gzip xxx
    • 解压缩：gunzip xxx.gz（或gzip -d
    • 其他
      • 后缀：.gz
      • 只可压缩文件，不可压缩目录
      • 压缩后不保留源文件
  • tar命令
    • 压缩：tar -zcvf 压缩后文件名 待压缩目录
      • z：打包的同时压缩
        • 若不带，则为.tar后缀，gzip后为.tar.gz
        • z可以替换为j，z-gzip、j-bzip2
      • c：创建一个压缩包，create，即打包或打包且压缩（看是否有z参数
      • v：显示详细信息
      • f：后加文件名，file
    • 解压缩：tar -zxvf xxx.tar.gz
      • x：解压缩
    • 其他
      • 操作对象为目录
      • 打包与压缩是不同的，一般为打包后再压缩
      • 后缀为.tar或.tar.gz，前者仅打包，后者打包后再压缩
  • zip命令
    • 压缩：zip -r 压缩后文件名 待压碎文件或目录
      • r：压缩目录
      • 格式类似tar，压缩后文件名在前
    • 解压缩：unzip xxx.zip
    • 其他
      • 改善gz，可保留源文件，可压缩目录
      • 压缩比不如gz，不常用
      • 后缀：.zip
      • Linux与windows通用
  • bzip2命令
    • 压缩：bzip2 -k xxx
      • k：压缩后保留原文件
      • 格式类似gz，直接加待压缩的文件
      • tar -jcvf xxx.tar.bz2 待压缩文件；j参数使用bzip2
    • 解压缩：bunzip2 -k xxx.bz2
      • k：解压后保留原压缩包文件
      • tar -jxvf xxx.tar.bz2；与tar相结合
    • 其他
      • 可与tar相结合
      • 效率较zip高，增强版
      • 后缀.bz2或.tar.bz2
  • 常见后缀
    • .gz
      • gzip
    • .zip
      • zip
    • .tar
      • tar
    • .tar.gz
      • tar
    • .bz2
      • bzip2
    • .tar.bz2
      • tar

文件搜索命令

  • find命令
    • find+搜索范围+匹配条件
    • 按文件名查找：find /etc -name init
      • 精确查找init文件名的文件
      • 利用匹配多个字符，模糊搜索，如init、init、init*
      • 利用?匹配单个字符，如：init???（init开头，后三个字符不确定
      • -iname：不区分大小写
    • 按文件大小查找：find /etc -size +204800
      • +n：大于、-n：小于、n：等于
      • 单位为数据块：1数据块=512字节=0.5k，1k=2数据块
    • 按文件所有者查找：find /home -user root
      • -group：根据所属组查找
    • 按时间属性查找：find /etc -cmin -5
      • 查找在5分钟内被修改过属性的文件和目录
      • -5:少于五分钟、+5超过五分钟
      • -amin：访问时间access
      • -cmin：文件属性被改的时间change
      • -mmin：文件内容被改的时间modify
    • 按文件类型查找：find /etc -type d
      • 查找此目录下所有目录类型的
      • d：目录、f：文件、l：软链接文件
    • 按i节点查找，-inum
      • ls -i：显示文件节点数
      • 可用于删除(或者其他操作)特殊文件名的文件（按文件名删除失败，按i节点数删除，find / -inum 1234 -exec rm {} \;
      • 可用于查看某文件的硬链接
        • 硬链接不可跨分区
        • 硬链接的文件i节点数是相同的
        • 得知某文件的i节点数，在同一分区find一下，可找到具有相同i节点数的文件
    • 匹配条件的与或关系-a、-o
      • find /etc -name init* -a -type f（多个查询条件通过a连接
      • -a：两个条件同时满足
      • -o：两个条件满足任意一个即可
    • 对搜索结果进行操作
      • find /home -name init -exec ls -l {} \ ;
      • 查找出文件并显示详细信息
      • -exec可以换为-ok，ok会询问是否确认（多用于删除操作时
      • -exec/-ok 命令 {} \;（对结果执行操作，注意空格
    • 其它
      • 搜索比较慢，locate快
      • 服务器高负载时尽量不使用find，其占用系统资源高
  • locate命令
    • 在文件数据库中查找文件
    • locate+文件名
    • locate -i + 文件名：不区分大小写
    • 数据库相关
      • 位置：/var/lib/mlocate/mlocate.db
      • updatedb命令手动实时更新数据库（系统会自动更新
      • /tmp目录：不在数据库的收录范围内
  • which命令
    • which + 命令名称
    • 搜索命令所在目录及别名信息
  • whereis命令
    • whereis+命令名称
    • 搜索命令所在目录及帮助文档目录
  • grep命令
    • grep + 选项+待搜索的字符串+文件
      • -i：不区分大小写
      • -v：排除指定的字符串
        • grep -v ^# xxx.conf
        • ^#：以#号开头
        • 排除掉以#开头的行，即去掉注释
    • 在文件内容中搜寻字符匹配的行并输出

文件处理命令

  • 命令格式
    • 命令 [-选项] [参数]
    • 个别命令的使用不遵循此格式
    • 有多个选项时，可写在一起
    • 简化选项与完整选项（-a与—all
  • 目录处理命令
    • ls命令
      • ls + 选项+文件或目录
        • -a：显示所有文件，包括隐藏
        • -l：显示详细信息
        • -d：查看目录属性（查看目录本身，而不显示目录下的
        • -h：以人熟悉的单位显示文件大小，如K、M，默认是字节B
        • -i：查看文件节点数，即文件ID号，唯一
      • 输出结果
        • 文件类型：-为二进制文件即普通文件、d为目录、l为软链接文件
        • u、g、o
        • r、w、x
    • mkdir命令
      • 创建新目录
      • mkdir -p +目录名
        • -p：递归创建
        • 可同时创建多个目录，空格分隔
    • cd命令：切换目录
      • cd + 目录
        • .当前目录
        • ..上级目录
        • ～家目录
        • -上次工作目录
    • pwd命令：显示当前目录
    • rmdir：删除空目录
    • cp命令：复制文件或目录
      • cp -rp +原文件或目录 +目标目录
        • r：复制目录
        • p：保留文件属性
      • 复制同时可以更改名称
    • mv命令：剪切文件、改名
      • mv +原文件或目录 +目标目录
    • rm命令：删除文件
      • rm -rf 文件或目录
        • r：删除目录
        • f：强制执行（避免删除前的询问
      • rf选项可用于删除目录，rmdir只可删除空目录
  • 文件处理命令
    • touch命令：创建空文件
      • 文件名支持特殊符号（比win多）但不建议使用
      • 不建议创建带空格的文件名（默认以空格为分隔，会创建多个文件
      • 若非要创建带空格的，加引号扩起来（有空格不方便，以后操作时都需要加引号
    • cat命令：显示文件内容
      • cat -n 文件名
        • -n：显示行号
      • 多用于查看小文件，内容少，否则不易观察
      • tac命令：以行为单位，反向显示文件内容
    • more命令：分页显示文件内容
      • 空格或f：翻页
      • 回车：换行
      • q或Q：退出
    • less命令：分页显示文件内容（可向上翻页
      • pageup：往上翻页
      • 上箭头：往上换行
      • /+字符串：搜索指定字符串（n往下找，
      • 其它同more
    • head命令：显示文件前面几行内容
      • head -n 20 test.txt
        • -n：指定行数
        • 默认前10行
    • tail：显示文件后面几行内容
      • -n：指定行数
      • -f：动态显示文件末尾内容（日志文件
  • 链接命令
    • ln：生成链接文件
      • ln -s 原文件 目标文件（s：创建软链接
      • 软链接后缀.soft、硬链接后缀.hard
    • 软链接
      • 类似win中快捷方式
      • 都是：lrwxrwxrwx
      • 文件小，仅是一个符号链接
      • ls -l查看时，箭头指向源文件
      • 源文件删除后，软链接失效
    • 硬链接
      • 除文件名及其目录，其它都一样
      • 效果等同与：cp -p + 同步更新
      • 源文件删除后，硬链接依然可使用（因其效果同拷贝
      • 硬链接文件同其源文件的i节点号一致，ls -i（故可以同步更新
      • 不能跨分区（软链接可以
      • 不可针对目录使用（软可以

权限管理命令

  • chmod：更改文件或目录权限
    • 只有root和文件所有者可更改
    • chmod u+r 文件或目录
      • u、g、o、a
      • +、-、=（如：g=rwx
      • r、w、x
      • 可同时加多个，如u+r,g+w（逗号分隔
    • chmod 421 文件或目录
      • rwx分别对应421
      • -R：递归修改
    • rwx权限
      • 文件
        • r：可查看文件内容
        • w：可修改文件内容
        • x：可执行文件
      • 目录
        • r：可以列出目录中内容
        • w：可在目录中创建、删除目录
        • x：可以进入目录
  • chown：更改文件或目录的所有者
    • chown +用户+文件或目录
    • root可更改，文件所有者不可更改
  • chgrp：更改文件或目录的所属组
    • chgrp +组名+文件或目录
    • root可更改，文件所有者不可更改
  • umask：显示、设置文件的缺省权限
    • umask或umask -S
      • S：以rwx形式显示
      • 不加S：显示掩码，如0022
    • 默认权限
      • 文件：rw-r–r–（新建文件默认无执行权限
      • 目录：rwxr-xr-x
    • 输出结果（不加S时
      • 0022
      • 0: 表示特殊权限
      • 022: 掩码，777-022=755，即rwxr-xr-x
    • 可设置默认权限
      • 如umask 023（只有后三位
      • 不建议更改
